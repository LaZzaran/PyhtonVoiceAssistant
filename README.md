# **Proje Açıklaması**



*Lazzaran - Türkçe Sesli Asistan
Lazzaran, çeşitli görevleri yerine getirebilen ve sesli komutlarla çalışan bir Türkçe dijital asistan projesidir. Bu asistan, kullanıcıdan sesli komutlar alarak belirli işlemleri gerçekleştirir ve metin tabanlı yanıtlar sağlar.*

## **Özellikler**

*Sesli Komut Tanıma: Mikrofon aracılığıyla Türkçe sesli komutları tanır ve metne dönüştürür.
Metinden Sese Dönüştürme: Yanıtları sesli olarak verir.
Hava Durumu: Belirtilen şehir için güncel hava durumu bilgilerini sunar.
Haberler: Güncel haber başlıklarını okur.
Not Alma: Sesli komutlarla notlar alır ve kaydeder.
Bilgisayarı Kapatma/Yeniden Başlatma: Bilgisayarı kapatma veya yeniden başlatma komutlarını yerine getirir.
Tarih ve Saat Bilgisi: Şu anki tarih ve saat bilgisini verir.
Google ve YouTube'a Erişim: Google ve YouTube'u açar.
Uygulama Başlatma: Bilgisayarda belirtilen uygulamayı başlatır.
Yapabileceği daha birçok şey var gemini ai yardımı ile de sizinle sohbet bile edebilir.*

**Gereksinimler:**

- Python 3.x
- speech_recognition kütüphanesi
- subprocess modülü
- turtle modülü
- google.generativeai kütüphanesi
- pyaudio kütüphanesi
- beautifulsoup4 kütüphanesi
- requests kütüphanesi
- pygame kütüphanesi

**Kurulum**

*Bu depoyu klonlayın veya indirin:*

`
git clone https://github.com/KULLANICI_ADI/REPO_ADI.git
cd REPO_ADI
`

*Gerekli Python kütüphanelerini yükleyin:*

`
pip install -r requirements.txt
`

*api_keys.py dosyasına API anahtarlarınızı ekleyin:*

`
gemini_api_keyim = "YOUR_GEMINI_API_KEY"
model_id = "YOUR_MODEL_ID"
haber_apikey = "YOUR_NEWS_API_KEY"
`

**Kullanım**

*Asistanı başlatın:*

`
python main.py
`
*Mikrofonu kullanarak komutlar verin ve asistanın yanıtlarını dinleyin.*
